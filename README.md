# VFD Simulator

An open-source simulator for industrial variable frequency drives. This project is intended as an educational tool and as part of a toolset to be used alongside [OpenPLC](https://github.com/Autonomy-Logic) and the [Open Industry Project](https://github.com/Open-Industry-Project).

The goal is to provide a platform where users can experiment with the fundamental parameters of a VFD and observe the resulting behavior of a motor.

## Features
- Motor physics simulation
- Modbus communication
- OPC UA integration
- Graphical interface
